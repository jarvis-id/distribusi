Sistem Dokumentasi Kinerja Digital Teknisi Lapangan

📖 Deskripsi Proyek

Distribusi (Laporan Dokumentasi) adalah aplikasi web Progressive Web App (PWA) generasi terbaru yang dirancang khusus untuk mempermudah teknisi lapangan dalam mendokumentasikan hasil kerja secara real-time. Dibangun oleh Stark Industries, aplikasi ini mengedepankan efisiensi, akurasi lokasi, dan kemudahan interaksi berbasis suara.

✨ Fitur Unggulan

🎙️ Jarvis Voice Assistant Pemandu suara cerdas yang membantu teknisi mengisi laporan tanpa harus terpaku pada layar secara terus-menerus.
📍 Geo-Watermark Precision Kamera cerdas yang menyematkan titik koordinat GPS, alamat real-time, dan peta lokasi langsung pada foto hasil kerja.
🔢 Dynamic Item Numbering Sistem pengurutan item otomatis yang memastikan laporan teknis tetap rapi meskipun ada penambahan atau penghapusan data.
💾 Smart Persistence Fitur draf otomatis yang menjamin data laporan tidak hilang meskipun browser tertutup secara tidak sengaja.
🔇 Mute Controller Kontrol penuh atas asisten suara dengan tombol mute mengambang yang responsif.
🖨️ Professional Export Cetak laporan ke format A4 yang sudah terstandarisasi untuk keperluan administrasi dan LKH.

🛠️ Teknologi Project

  • Core Interface: HTML5, CSS3 Modern, Vanilla JavaScript (ES6+).
  • Intelligence: Speech Synthesis API (Jarvis Voice Engine).
  • Imaging: HTML5 MediaDevices API with Custom CSS Filter Overlay.
  • Mapping: Yandex Static Maps & Nominatim Geocoding Integration.
  • Infrastructure: Serverless PWA with LocalStorage Persistence.

📲 Pasang Aplikasi

Gunakan aplikasi Distribusi dengan akses cepat & fitur offline.

📋 Panduan Instalasi per Browser

🌐 Google Chrome
1
Desktop: Klik ikon Install (monitor kecil) di kanan kolom alamat URL ATAU klik Titik Tiga (⋮) > Save and Share > Install page as app.
2
Android: Klik Titik Tiga (⋮) > Install app.
💻 Microsoft Edge
1
Klik Titik Tiga (...) di pojok kanan atas.
2
Pilih Apps > Install this site as an app.
🍎 iPhone / Safari (iOS)
1
Klik ikon Share (kotak dengan panah ke atas) di bagian bawah.
2
Gulir ke bawah dan pilih Add to Home Screen.
🌌 Samsung Internet
1
Klik Garis Tiga (≡) di pojok kanan bawah.
2
Pilih Add page to > Home screen.
🦊 Mozilla Firefox (Hanya Mobile)
1
Klik Titik Tiga (⋮) di pojok kanan bawah.
2
Pilih Install atau Add to Home Screen.
⚠️ Catatan:
Untuk pengalaman terbaik, gunakan Google Chrome, Microsoft Edge, atau Safari. Browser lama seperti Internet Explorer tidak mendukung fitur ini.
