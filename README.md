# 📝 Distribusi - Stark Industries
**Sistem Dokumentasi Kinerja Digital Teknisi Distribusi dan Jaringan**

---

### 📖 Deskripsi Proyek
**LapDok** adalah aplikasi web *Progressive Web App* (PWA) yang dirancang oleh **Stark Industries** untuk membantu teknisi lapangan dalam mendokumentasikan hasil kerja secara digital dengan akurasi tinggi.

### ✨ Fitur Unggulan
- **🎙️ Jarvis Voice Assistant**: Pemandu suara cerdas untuk pengisian laporan.
- **📍 Smart GPS Camera**: Watermark lokasi (GPS & Alamat) langsung pada foto.
- **🔢 Dynamic Numbering**: Penomoran item otomatis yang selalu urut.
- **💾 Local Persistence**: Sistem draf aman tanpa database server.
- **🖨️ Professional Print**: Format cetak A4 otomatis untuk laporan resmi.

### 🛠️ Teknologi
- HTML5, CSS3, JavaScript (ES6+)
- Browser Speech API & Geolocation API
- OpenStreetMap & Yandex Maps API
- PWA (Service Workers & Manifest)
